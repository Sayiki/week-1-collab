# MakanMana

### **Tagline**
> Solusi anti-bingung cari makan murah di sekitar kita.

## **Masalah yang Ingin Diselesaikan**
Banyak orang sering membuang waktu 15-30 menit hanya untuk berdiskusi "makan di mana?" karena pilihan yang terlalu banyak atau ketidakpastian harga di lokasi.

## **Background**
Keterbatasan anggaran dan waktu menuntut adanya daftar kurasi tempat makan yang ramah kantong namun tetap berkualitas. Aplikasi ini dibuat untuk mempermudah siapa saja menemukan pilihan kuliner terbaik tanpa harus bingung memilih.
